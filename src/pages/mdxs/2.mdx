# Hello MDX 2

## What is this

This is a pre-configured Next.js boilerplate which contains TailwindCSS, @next/mdx and next-mdx-remote and useful mdx plugins.

## Features for

- Syntax highlight with `highlight.js` or `prism`
- Display React components in Markdown
- Automatically add `id` to headings (H1, H2,...) by `rehype-slug`
- Automatically add link icon when mouse over headings by `rehype-autolink-headings`

## Syntax highlight

```ts
const message: string = "Hello Markdown";
const num: number = 1234567890;
```
